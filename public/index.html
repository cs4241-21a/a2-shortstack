How to use

Todoish tries to emulate a Todoist todo list. You can set a task's priority, due date, etc.

Importance is a derived field in a range from 0 to 100. The higher the importance, the more important that task is.

Create new tasks with the panel on the side. Edit, Complete, and Delete Tasks with the inline table.
