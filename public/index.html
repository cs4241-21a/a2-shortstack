Homework Tracker!

Do you have lots of schoolwork? Use this application to determine how many days you have left to submit your assignment!

How this works: Type in the class and assignment you want to keep track of, select the drop down for whether or not this assignment is complete, select the due date, and click the submit button. Note: If you do not select a due date, you will not be able to calculate the days left.
